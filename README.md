# Flutter Calculator App

## Description
A simple calculator app built with *Flutter* and the *math_expressions* library for evaluating mathematical expressions. It features a clean UI with button-based input and real-time calculation support.

## Widgets Used
* *MaterialApp*: Sets up the app structure and defines the home screen.
* *Scaffold*: Provides the basic layout with an app bar and body.
* *AppBar*: Displays the app title ("Calculator").
* *Container*: Holds and styles the display text with padding and alignment.
* *Text*: Shows the current input or result on the screen.
* *Divider*: Separates the display area from the button grid.
* *Row*: Arranges buttons horizontally in each row.
* *ElevatedButton*: Interactive buttons for digits, operators, and actions (*e.g., "C", "=", ".", etc.*).
